의심 병명 목록

고혈압

: 혈압이 여러 원인으로 인해 높아진 상태를 의미합니다.

근육 긴장성 두통

: 두통이 머리에 띠를 두른 것처럼 양측성으로 나타나는 가장 흔하고, 일반적으로 스트레스, 피로, 수면 부족 등의 요인에 의해 발생하는 가벼운 두통입니다.

금단현상

: 특정 약물이나 대상, 행위를 충동적, 습관적으로 하게 되며, 이를 중단할 경우 정신적, 물리적 여러 증상이 나타나는 것을 의미합니다.

뇌경색

: 뇌혈관이 막혀 뇌의 일부가 손상되는 질환을 의미합니다.

기뇌증

: 공기가 있으면 안 되는 뇌에 어떤 요인으로 인해 뇌에 공기가 들어가 뇌혈관이 막히거나 뇌조직이 압박되어서 발생하는 질환입니다.

뇌졸증

: 뇌의 일부분에 혈액을 공급하는 혈관이 막히거나(뇌경색) 터짐(뇌출혈)으로써 그 부분의 뇌가 손상되어 나타나는 신경학적 증상을 말합니다.

미니 뇌졸증

: 뇌졸중 증상이 본격적으로 나타나기 24시간 전에 아주 드물게 나타나는 전조 현상입니다. (사지부분 마비, 언어장애, 인지장애)

저혈압

: 혈압이 여러 원인으로 인해 낮아진 상태를 의미합니다.
